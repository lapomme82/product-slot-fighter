## 로컬 간 작업 규칙

작업 시작 전:

```bash
git pull --ff-only
npm ci
```

작업 종료 전:

```bash
npm test
npm run build
git status
```

변경 사항이 있으면 의미 단위로 커밋한 뒤 GitHub에 푸시합니다.

```bash
git add <changed-files>
git commit -m "Describe the completed work"
git push
```

다른 로컬에서 이어 작업할 때는 먼저 `git pull --ff-only`로 최신 커밋을 받은 뒤 시작합니다.
